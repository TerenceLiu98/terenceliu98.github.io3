New Blog

2019-08-18

What is this?

This is my new blog. I am tired of using HTML and writing the CSS. I am now using Hugo and Github Pages.

What is the difference

This is a different experience than the normal Hugo + Github Pages, since this is also powered by RMarkdown, which means I can writing blog in Rstudio and also Rmarkdown is friendly to our statistics student because of the R. For instance:

library(ggplot2)
ggplot(diamonds, aes(x = carat, y = price, colour = clarity)) +
  geom_point() +
  xlab("Carats") + ylab("Price") + 
  ggtitle("Diamonds")

I can easily draw a plot by using Rmd. In the above plot, I draw it by using the famous library: ggplot; what’s more, it can eaisly for us to add plug-in in our post, like, twitter, youtube and etc.:

blogdown::shortcode("youtube", "CjTLN-FXiFA")

It is a ‘sharp knife’ for our “bvlogger”, I means blogger + vlogger actually.

Also, since it combined with hugo, I can easily push my post on GitHub and use the GitHub Pages service, which means I can save some server money.
