What is this?

This is my new blog. I am tired of using HTML and writing the CSS. I am now using Hugo and Github Pages.

What is the difference

This is a different experience than the normal Hugo + Github Pages, since this is also powered by RMarkdown, which means I can writing blog in Rstudio and also Rmarkdown is friendly to our statistics student because of the R. For instance:

# Define 2 vectors
cars <- c(1, 3, 6, 4, 9)
trucks <- c(2, 5, 4, 5, 12)

# Graph cars using a y axis that ranges from 0 to 12
plot(cars, type="o", col="blue", ylim=c(0,12))

# Graph trucks with red dashed line and square points
lines(trucks, type="o", pch=22, lty=2, col="red")

# Create a title with a red, bold/italic font
title(main="Autos", col.main="red", font.main=4)

I can easily draw a plot by using Rmd.
