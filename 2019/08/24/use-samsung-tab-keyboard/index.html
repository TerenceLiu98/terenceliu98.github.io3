use samsung tab keyboard

2019-08-24

如题所示，我现在就在使用三星的 tab S3 官配的磁吸键盘进行写作。写作速度缓慢，键盘的尺寸是一个影响因素，更多的是因为在使用远程桌面，而且延时还不算低。

为什么会购买这个键盘

原因很简单： 1. 我有一个三星 tab S3 （这个假期买的） 2. 我没有带罗技的 K480 回家

其实第二个原因是不成立的，因为 K480 十分的不便携，非常的大不说，重量也感人。另， 我也想试试磁吸键盘到底怎么样，身边太多的人在使用 iPad + keyboard + Apple Pencil，可能我是为数不多在使用全三星套装的人 (Samsung S8 + tab S3 + keyboard + Spen)。 嗯，非常不政治正确，没有购入华为的 M6（虽然真的有想过）

体验如何

到目前为止，我还是有些不适应，键帽确实小，对于我这种 MX Master2 才算刚刚好的手来说真的小，不过也还行，因为他的键帽反馈可以接受（跟 MBP 2015 retina 比较像， 都是比较直上直下的）如果这种键盘可以称得上有反馈的话；绝对是比 ipad 的键盘好，毕竟是硬的（谁不喜欢硬的呢）

另外，这个 9.7 寸屏幕对于写写文章的人来说已经不错了，偶尔改改代码也是极好的。甚至可以分屏使用， 比如我复制 yihui 写的这段代码：

set.seed(123)
# 真实模型
x = sort(runif(100))
y = 2 + 1 * x + 4 * (x > 0.5) + 3 * (x - 0.5) * (x > 0.5) + rnorm(100)
par(mar = c(4, 4, 0, 0), family = "serif", mgp = c(2, 1, 0))
plot(x, y, pch = 20, col = "darkgray")
# 斜率不同，截距限制
fit1 = lm(y ~ 1 + x + I((x - 0.5) * (x > 0.5)))
lines(x, fitted(fit1), lwd = 2, col = 2)
# 斜率不同，截距也不同
fit2 = lm(y ~ 1 + x + I(x > 0.5) + I((x - 0.5) * (x > 0.5)))
lines(x, fitted(fit2), lwd = 2, lty = 2)

总结

To sum up, 这个键盘给我的体验还是不错的，希望能在新学期更大的效用， 毕竟算是一个轻薄平板 + 台式；至于我的 MBP， emmm，因为某些不可描述原因，只能当作台式二号机
